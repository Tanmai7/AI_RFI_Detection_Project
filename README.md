# AI RFI Detection Project

Repository structure:

- data/raw/      - raw input data
- data/processed/ - cleaned/processed data
- notebooks/     - exploratory and training notebooks
- models/        - saved models
- dashboard/     - dashboard app files
- results/       - evaluation outputs and artifacts
- report/        - final reports and docs

# AI-Based Radio Frequency Interference Detection and Classification

This project focuses on detecting and classifying Radio Frequency Interference (RFI) using Machine Learning techniques.

## Key Features
- RFI Detection (Yes / No)
- RFI Classification (Narrowband, Broadband, Impulsive)
- Signal Visualization (Time, Frequency, Spectrogram)
- Machine Learning Model Comparison
- Interactive Dashboard

## Tools Used
- Python
- NumPy, Pandas, SciPy
- Scikit-learn
- Streamlit
- GitHub
